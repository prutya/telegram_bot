# telegram_bot

Telegram Bot API client in Crystal

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     telegram_bot:
       github: prutya/telegram_bot
   ```

2. Run `shards install`

## Usage

```crystal
require "telegram_bot"
```
